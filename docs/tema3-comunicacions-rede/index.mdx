# Tema 3: Programación de Comunicacións en Rede con Sockets

## Contidos

Neste tema estudiaremos a programación de aplicacións de rede:

- Modelo TCP/IP
- Sockets TCP e UDP
- A clase `Socket` e `ServerSocket`
- DatagramSocket para UDP
- Enderezos IP e portos
- Protocolos de rede
- Cliente-servidor

## Obxectivos

Ao finalizar este tema, o alumno será capaz de:

- Desenvolver aplicacións cliente-servidor
- Implementar comunicacións TCP e UDP
- Crear protocolos de comunicación personalizados
- Xestionar conexións de rede
