# Tema 2: Programación Multifío

## Contidos

Neste tema profundizaremos na programación con fíos (threads):

- Conceptos de fío e multifío
- Estados dun fío
- Creación de fíos en Java
- A clase `Thread` e a interfaz `Runnable`
- Métodos principais: `start()`, `join()`, `sleep()`, `interrupt()`
- Fíos daemon
- Sincronización de fíos

## Obxectivos

Ao finalizar este tema, o alumno será capaz de:

- Crear e xestionar fíos en Java
- Implementar aplicacións multifío básicas
- Controlar o ciclo de vida dos fíos
- Resolver problemas de concorrencia
