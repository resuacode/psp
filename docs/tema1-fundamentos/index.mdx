# Tema 1: Fundamentos da Programación Concorrente e Procesos

## Contidos

Neste tema estudiaremos os conceptos fundamentais da programación concorrente:

- Conceptos de proceso e multitarea
- Estados dun proceso
- Planificación de procesos
- Creación e xestión de procesos en Java
- A clase `ProcessBuilder`
- Comunicación entre procesos

## Obxectivos

Ao finalizar este tema, o alumno será capaz de:

- Comprender os conceptos básicos da programación multitarea
- Crear e xestionar procesos desde Java
- Implementar comunicación básica entre procesos
