# Tema 4: Xeración de Servizos en Rede e Seguridade

## Contidos

Neste tema estudiaremos o desenvolvemento de servizos de rede e técnicas de seguridade:

- Arquitecturas cliente-servidor
- Servizos web
- APIs REST
- Servidores web
- Criptografía básica
- Autenticación e autorización
- Certificados dixitais
- SSL/TLS

## Obxectivos

Ao finalizar este tema, o alumno será capaz de:

- Deseñar e implementar servizos de rede escalables
- Desenvolver APIs REST
- Configurar servidores web básicos
- Implementar mecanismos de seguridade básicos
- Utilizar criptografía en aplicacións Java
